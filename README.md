# gameboy emulator

a work-in-progress gameboy emulator written in c. worked on whenever i have the time.

## what's here

- cpu core (z80 instruction set)
- register and flag operations
- memory management
- basic instruction implementations

## status

in progress. lots to do.

## building

```bash
gcc -o emulator *.c
```

just a simple c project, nothing fancy.
